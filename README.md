# ResistorLabels
This script generates Labels for Resistor zip bags.

It is meant for AVERY 5260 Labels and 7x10cm (3"x4") zip bags.

The generated labels include:
  * Resistor value
  * 4- and 5-band color codes
  * 3- and 4-digit smd codes
  * EIA-96 smd code
